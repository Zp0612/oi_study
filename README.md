## Commit 规范

```
solve: <题号> <题名>          # 做新题
improve: <题号> <改进点>      # 优化解法
template: <算法名>            # 添加模板
note: <内容>                  # 学习笔记
```

## 常用命令

```bash
git add .                              # 暂存所有改动
git commit -m "solve: P1001 A+B"       # 提交
git push origin main                   # 推送
```
