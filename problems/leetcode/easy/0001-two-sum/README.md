#1、两数之和

##1. 暴力法 O(n^2)：
双循环，两两相加，判断是否等于target：注意外层j循环的起始位置

##2.哈希表 O(n)：
哈希表 unordered_map
    创建一个哈希表，将nums[i]作为key，i作为value，然后遍历nums，判断target - nums[i]是否存在于哈希表中，如果存在，则返回[value, i]，如果遍历结束，则返回[]
注意：哈希表查找的时间复杂度是O(1)
     为什么先查找、后存入？
     如果先把当前数字存进去再查找，那遇到 target = 6、当前数是 3 的情况，会直接匹配到自己，用了同一个元素两次，就违反题目规则了。先查后存就能保证：找的永远是之前遍历过的数，不会和自己相配对。

本题考点：
  哈希表的基础应用
  空间换时间的算法优化思想
  数组遍历与边界处理
